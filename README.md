# Interpreting and de-noising genetically engineered barcodes in a DNA virus

This is the code used to analyze NGS data for the paper "Interpreting and de-noising genetically engineered barcodes in a DNA virus", Blois et al. The analysis is primarily in R.

The `stock` directory contains a short notebook and scripts for analyzing the data for the plasmid library, ligated virus library, and the virus library. The `controls` directory contains the script used to analyze and generate figures for the known plasmid controls generated to check the computations.
